HTML 筆記 - HTML 基本環境建立
HTML
建立日期 : 2019/08/25

我們在撰寫 HTML 的時候，基於 HTML 的標準，在撰寫時必須符合一些規範才有辦法讓檔案的內容呈現在瀏覽器上面，而這個撰寫規範如下。

  • <!DOCTYPE html>是在告訴瀏覽器要用 這是一個 HTML5 的檔案，就如同 word 一樣，如果是 2003 的版本副檔名會標示 .doc，如果是 2007 以上的版本，副檔名會標示 .docx，所以要針對不同的檔案用相對應的方式去編譯。
  • 內容會拆分為 head 和 body
    • head 是用來放置網頁的一些資訊
    • body 是用來放置網頁的內容
