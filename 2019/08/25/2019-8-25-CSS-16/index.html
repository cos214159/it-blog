CSS 筆記 - CSS 在不同瀏覽器的支援度
CSS
建立日期 : 2019/08/25

CSS 並非在每一個瀏覽器都可以運行，有些是不支援的，但我們可以加上一些前綴詞讓瀏覽器可以支援這個 CSS 屬性，而前綴詞如下

  • Android: -webkit-
  • chrome: -webkit-
  • Firefox: -moz-
  • IE: -ms-
  • ios: -webkit-
  • Opera: -o-
  • safari: -webkit-
