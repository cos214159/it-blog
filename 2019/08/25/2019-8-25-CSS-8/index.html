清除浮動

float 我覺得比較麻煩的地方是因為 float 的區塊本身並不具有高度，所以如果要在浮動區塊之後再繼續加上內容，我們就要針對浮動區塊清除浮動，否則後面的內容會被浮動區塊的內容蓋住。

而清除浮動的語法是 clear: both
