# table

表格就跟 word 的表格很像，會有表頭以及表格內容，相對應的標籤結構如下

``` HTML
<table class="table">
  <tr class="table__row">
    <th>會員編號</th>
    <th>姓名</th>
    <th>電話</th>
  </tr>
  <tr>
    <td>0</td>
    <td>AAA</td>
    <td>0912345678</td>
  </tr>
</table>
```

基本上外層一定要有一個 table 標籤，而內層如果是表格標題要使用 `th`，而如果是表格的內容就要用 `td` 包住
