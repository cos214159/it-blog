舉個例子，像連結標籤 `a` 當使用者在操作時，我們會想給使用者不一樣的操作回饋，像滑鼠滑過去時會變色、點擊時會出現下底線 ... 等，以下為範例

## 滑鼠移過去時的效果
``` css
a:hover {
  color: red;
  font-size: 50px;
}

```

## 點擊時的效果
``` css
a:active {
  color: yellow;
  font-size: 50px;
}
```
