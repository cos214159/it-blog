Electron 有提供關於電池狀態的 api。

``` js
const electron = require('electron')

electron.powerMonitor.on('resume', e => {
  if(!mainWindow) createWindow()
})

electron.powerMonitor.on('suspend', e => {
  console.log('Saving some data')
})

```
