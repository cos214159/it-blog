漸層色預設是由上到下 
``` CSS
.box1 {
  width: 500px;
  height: 200px;
  background: linear-gradient(#000, blue);
}
```

如果想要改變方向的話可以多加上一些方向性的屬性
```CSS
.box2 {
  width: 500px;
  height: 200px;
  background: -webkit-linear-gradient(right, #000, blue);
}
```
