## 新增套件

```
yarn add <package name>
```

在新增完套件之後，yarn 會將套件安裝到 `node_modules` 資料夾底下，以及會額外新增 `package.json` 和 `yarn.lock`

## 刪除套件

```
yarn remove <package_name>
```

在刪除套件之後，yarn 會將該套件以及相關模組從 `node_modules` 裡面移除
