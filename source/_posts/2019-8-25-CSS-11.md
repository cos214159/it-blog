如果想要在畫面上畫出一個圓形，我們可以使用 CSS3 提供的 `border-radius` 功能，把值設定成 50%

``` CSS
.box {
  width: 70px;
  height: 70px;
  background: #000;
  border-radius: 50%;
  border: 10px solid red;
}
```

或者是也可以針對四個角的弧度各別調整
``` CSS
.box1 {
  margin-top: 30px;
  width: 70px;
  height: 70px;
  background: #000;
  border-radius: 50px 0 0 50px;
  border: 10px solid red;
}
```
